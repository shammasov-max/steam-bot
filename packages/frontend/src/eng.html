В этом разделе описаны все функции данной программы, и оставлены контакты разработчика.

Авторизация

Для авторизации необходимо загрузить аккаунты вида log:pass во вкладке Import accounts, после чего нажать кнопку Login.

Так же, можно авторизировать аккаунты используя прокси, импортировав их перед авторизацией во вкладке Import proxy. На 1 аккаунт нужна 1 прокси, можно дублировать прокси в импорте, если не хватает количества.

Чтобы привязать определенную прокси к аккаунту - нужно импортировать аккаунт вида: login:password - proxy

Загрузка прокси в формате host:port:log:pass

После загрузки всех аккаунтов в консоли появится сообщение "All accounts logged successfully"

Рекомендую указать папку с мафайлами через настройки, чтобы не вставлять их каждый раз после обновления в папку с программой.

Don't remember machine - Функция, которая позволяет не запоминать ваши ssfn файлы.

Fast Auth - Функция, которая позволяет быстро авторизировать аккаунты. Рекомендуется использовать в следующих случаях:

  • 1. Вы используете прокси
  • 2. У вас мало аккаунтов, и нету прокси ( до 100 )
  • 3. У вас аккаунты без стим гуарда
  • 4. Вам не нужно вводить гуард код вручную

Connection method - Способ соединения с серверами стима.

  • 1. Auto - Автоматическая настройка
  • 2. Method 1 - Если у вас нет никаких проблем с прокси\интернетом
  • 3. Method 2 - Если у вас проблемы с интернетом\прокси.
Если вас не беспокоят вылеты интернета\прокси - можете ставить Auto.

Настройки

1. Templates - Это шаблоны для отправки заготовленных сообщений. При создании шаблонов в диалоге появятся кнопки для отправки заготовленных сообщений.

Шаблоны имеет следующий вид:

  • Имя_шаблона|Текст шаблона;
  • Имя_шаблона|Текст шаблона1|Текст шаблона2|Текст шаблона3|Текст шаблона4;
  • Имя_шаблона|Текст шаблона template_color=red;
  • Имя_шаблона|Текст шаблона template_link;
  • Имя шаблона|Текст шаблона template_username;
  • Имя шаблона|Текст шаблона template_invite_link;
  • Имя шаблона|Текст шаблона template_dearest_item_IDигры
  • и тд.

template_color - задает определенный цвет для кнопки (Поддерживается любой формат цвета: red, #dcdcdc, #000)

template_link - позволяет мгновенно отправить шаблон, сделано для ссылок ( Можно использовать не только для ссылок ), чтобы они не печатались сильно долго, а отправлялись сразу, как буд то вы их вставили через CTRL + V

template_username - Позволяет подставлять никнейм пользователя в сообщение, например: Hi, template_username, how are you ?. При отправке сообщения пользователю с никнеймом ino, сообщение примет следующий вид: Hi, ino, how are you ? (Работает так же для spam all, автоотправка отправка сообщения новому другу )

template_invite_link - Аккаунт отправляет свою ссылку на приглашение в друзья, для этого, перед использованием этой функции ссылку нужно создать через actions - invite friends - create invite links. Ссылка создается один раз и не имеет ограничений как в стиме. Данная функция работает во всех функциях автоматической отправки сообщений: шаблонах, spam all, wall spamm, сообщение новому другу и т.д.

template_dearest_item_IDигры - Подставляет название самого дорогого предмета из выбранной игры в сообщение. Если инвентарь не загружен или отсутствует, это слово будет пропущено. Пример: Hi, i like your template_dearest_item_730

template_image=путь_к_картинке - Можно использовать в шаблонах и в рассылке личных сообщений, чтобы отправить картинку со своего ПК.

Если вы разделяете в одном шаблоне сообщения через | - то они будут отправляться друг за другом.

2. Message for new friends - Сообщение которое будет автоматически отправляться новым друзьям когда они принимают заявки в друзья, для этого нужно включить галочку Auto message for new friend

При разделении сообщений через ; будет выбрано случайное сообщение. Так же как и в шаблонах, при разделении сообщений через | они будут отправляться друг за другом.

3. Auto message timeout (sec) - Количество секунд, через которое будет отправлено сообщение новому другу.

4. Auto template timeout - При включении данной опции, задержка на отправку шаблона будет рассчитываться автоматически от длины шаблона, при этом, вашему собеседнеку на протяжении всей задержки показывает что вы печатете сообщение.

Если данная опция не включена - шаблон любой длины будет отправляться с одинаковой задержкой (4сек)

5. Auto template timeout for 1 symbol (milisec) - Количество милисекунд на 1 символ при отправке шаблона с автоматическим рассчитыванием задержки.

6. Paste template to input - При включении этой опции шаблон будет вставляться в поле для ввода сообщения, вместо отправки. Данная функция нужна если вам нужно редактировать шаблоны под разные ситуации.

7. Sound - Вкл./Откл. звук уведомлений

8. Notification - Вкл./Откл. системные уведомления. Уведомления кликабельные как в телеграмме.

Если вам не приходят уведомления, но опция включена - запустите программу в режиме совместимости с Win 7

9. Save deleted friends - Cохраняет друзей в программе после удаления или блокировки.

10. Auto pin users with unread messages - При авторизации аккаунтов автоматически закрепляет пользователей, от которых висят непрочитанные сообщения.

11. Auto install update - При включении данной опции обновления будут скачиваться, и устанавливаться автоматически ( Только для пользователей Windows )

12. Path to maFiles - Возможность указать папку с мафайлами вручную. Рекомендуется использовать данную функцию, чтобы после обновлений программа автоматически считывала мафайлы из указанной папки.

13. Background color - Цвет фона программы. (Поддерживается любой формат цвета: red, #dcdcdc, #000)

14. Chat background color - Цвет фона чата программы. (Поддерживается любой формат цвета: red, #dcdcdc, #000)

15. Account name color - Цвет аккаунта в чате. (Поддерживается любой формат цвета: red, #dcdcdc, #000)

16. Open chat color - Цвет, которым подсвечивается открытый диалог. (Поддерживается любой формат цвета: red, #dcdcdc, #000)

17. template_domain - Это переменная, вместо которой в программе будет подставляться ваш домен, а именно: При спаме по стенкам, при спаме по ЛС, при отправке сообщения новым друзьям и при отправке шаблонов.

Чтобы использовать переменную template_domain - создайте переменные которые заканчиваются на _domain, например:

csgo_domain|https://csgodomain.com;

dota_domain|https://dotadomain.com;

Работают по тому же принципу что и обычные шаблоны

18. Get user info ( lvl, country ) - Включение и отключение получении информации о друге (лвл и страна)

Опции Check Trade и ниже - опции которые будут отображаться при нажатии ЛКМ на аватар друга, при выключенных галочках опции не отображаются.

19. Auto boost hours at startup - Автоматическая накрутка часов при авторизации аккаунтов

20. Deleted friend background - Цвет, которым будут подсвечены удаленные друзья

Games prices

Функция для подсчета цен инвентарей ваших друзей. Те игры которые будут включены - будут проверяться. При желании, можно отключить полностью проверку цен инвентаря, или оставить только те игры, которые вас интересуют.

Friends

1. Hide offline - Прячет или показывает оффлайн друзей.

2. Search by name - Поиск по никнейму в списке друзей.

3. Hide by name - Данная функция не показывает людей, у которых в нике есть определенный символ.

4. UP - Функция для скролла списка друзей в самый верх списка

5. DOWN - Функция для скролла списка друзей в самый низ списка

6. UNPIN ALL - Открепить всех друзей

Чтобы просмотреть список доступных действий аккаунта - нужно нажать ПКМ на логин аккаунта

1. Pending Invites - Список входящих и исходящих заявок в друзья.

2. Account page - Открыть аккаунт в браузере по умолчанию.

3. Open Steam (Browser) - Открыть аккаунт в браузере программы. При этом аккаунт будет авторизованный в стиме, и можно выполнить любое нужное действие.

4. Open a website (Browser) - Открыть любую ссылку в браузере программы.

5. Get Guard Code - Сгенерировать гуард код для входа ( Только для аккаунтов с maFile )

6. Remove account - Удалить аккаунт из программы. При удалении аккаунта так же удаляются все друзья связанные с этим аккаунтом. Данную функцию можно использовать для очистки забаненных аккаунтов, или перезагрузить аккаунт в программе.

7. Statistic - Просмотр статистики о количестве принятых заявок в друзья и удаленных друзей. Во время работы программы записывается каждое добавление и удаление в друзья, вы можете посмотреть статистику за каждый день по добавлению и удалению, и сделать какие то выводы о своей работе. Каждый месяц статистика очищается.

Можно закрепить и открепить любого друга нажав на него ПКМ

Если аккаунт подсвечивается красным цветом - значит он заблокирован. Вы можете удалить его из программы нажав ПКМ на логин аккаунта - Remove account

Actions

1. Spam all - Функция для рассылки сообщений всем друзьям на выбранных аккаунтах. Выбрав соотв. галочки с онлайном сообщения будут отправлены только людям у которых онлайн равен выбранному.

Так же как и в шаблонах можно указывать случайные сообщения через ; и отправлять сообщения друг за другом разделяя их через |

Don't send spammed users - При включении данной опции сообщения не будут отправлены тем, кому уже были отправлены через функцию Spam All

2. Invite Friends - Функция для рассылки приглашений в друзья с выбранных аккаунтов. Для приглашений в друзья нужно загрузить список steamid64.

  • 2.1. Delete Duplicates - Функция по удалению одинаковых стимайди со списка
  • 2.2. Use Blacklist - При включении данной опции заявки не будут отправлены тем людям, которые содержаться в черном списке. При отправки заявок, принятии в друзья, удалении и блокировке люди автоматически попадают в этот список.
  • 2.3. Enter invite time - Возможность указать время, в которое будет начат инвайт. Время нужно указывать в таком формате: 20:44
  • 2.4. Enter reinvite hours ( 0 - for disable ) - Возможность добавлять в друзья по кругу каждые N часов, 0 чтобы отключить.
  • 2.5. Enter invite count - Количество заявок которое будет отправлено с одного аккаунта. По умолчанию 30, т.к. в стиме лимит 30 заявок в друзья с одного аккаунта в неделю.

3. Clear Invites - Функция по удалению отправленных инвайтов в друзья с выбранных аккаунтов. Данная функция имеет возможность удалять определенный процент заявок, или удалять заявки друзей, которые оффлайн более опредленного количества дней.

4. Clear Friends - Функция по удалению друзей с выбранных аккаунтов. Удаляет друзей которые оффлайн более опредленного количества дней, либо null, чтобы удалить всех

5. Hour Boost - Функция по накрутке часов на выбранных аккаунтах. Есть возможность играть в несколько игр одновременно. Для этого нужно прописать appid игры. Например: 730,440,570

6. Set Online state Функция для выставления онлайн статуса на выбранных аккаунтах и UI мода.

7. Wall Spam - Функция для спама по стенкам с выбранных аккаунтов. При большом количестве аккаунтов необходимы прокси

8. Delete Hold Users - Функция по удалению друзей с трейдхолдом на выбранных аккаунтах

9. Edit Profile - Функция по изменению настроек аккаунта, настроек приватности и аватара. При разделении данных через ; будут выбираться случайные.

10. Pin by online state - Функция для автоматического закрепления друзей с выбранных аккаунтов по выбранному онлайн статусу.

11. Clear name history - Функция по очистке истории ников на выбранных аккаунтах.

12. Community tasks - Функционал по прокачке уровня стим через прохождение заданий сообщенства. Дання функция проходит 17 заданий, если у вас аккаунт без лимита и с вкл. стим гуардом. 18-е задаение вы можете пройти воспользовавшись функцией set avatar
